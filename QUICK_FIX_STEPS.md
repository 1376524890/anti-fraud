# 快速修复步骤 - 分类筛选问题

## 🚀 立即执行（5分钟）

### 1️⃣ 部署云函数（必须）
在微信开发者工具中：
- 右键 `cloudfunctions/getArticles` → "上传并部署：云端安装依赖"
- 右键 `cloudfunctions/checkArticleCategories` → "上传并部署：云端安装依赖"
- 右键 `cloudfunctions/fixArticleCategories` → "上传并部署：云端安装依赖"

### 2️⃣ 运行诊断
1. 打开"云开发控制台" → "云函数" → `checkArticleCategories`
2. 点击"测试"，输入 `{}`，点击"运行测试"
3. **记录返回结果**（特别是 `categoryStats` 和 `unexpectedCategories`）

### 3️⃣ 自动修复（推荐）

#### 方法A：使用自动修复云函数（快速）
1. 打开"云开发控制台" → "云函数" → `fixArticleCategories`
2. 点击"测试"，输入 `{}`（试运行模式，不修改数据）
3. 查看返回结果，确认需要修复的文章
4. 如果确认无误，输入 `{ "dryRun": false }` 正式修复
5. 等待修复完成

#### 方法B：手动修复（如果需要）

**如果 `unexpectedCategories` 不为空：**
说明数据库中的分类名称与前端不匹配

示例问题：
```json
"unexpectedCategories": ["刷单返利 ", " 虚假投资理财", "冒充电商客服。"]
```

**修复方法：**
1. 云开发控制台 → 数据库 → articles
2. 点击"查询"，输入：`{ "category": "刷单返利 " }`（注意空格）
3. 批量选中 → "批量编辑" → 改为 `"刷单返利"`（无空格）
4. 对每个错误的分类重复此操作

**如果 `articlesWithoutCategory` 不为空：**
说明有文章缺少 category 字段

**修复方法：**
1. 云开发控制台 → 数据库 → articles
2. 找到对应的文章（根据 `_id`）
3. 点击"编辑"，添加 `category` 字段
4. 值选择：`刷单返利`、`虚假投资理财`、`虚假购物服务`、`冒充电商客服`、`虚假贷款`、`虚假征信`、`冒充领导熟人`、`冒充公检法`、`网络婚恋交友`、`网游虚假交易`

### 4️⃣ 清除缓存并测试
1. 微信开发者工具 → "清缓存" → "清除数据缓存"
2. 重新编译
3. 依次点击每个分类标签，确认显示正常

## ⚠️ 常见错误

### 错误 1：分类名称有空格
❌ 数据库：`"刷单返利 "` （后面有空格）  
✅ 应该是：`"刷单返利"`

### 错误 2：中英文标点混用
❌ 数据库：`"冒充电商客服。"` （中文句号）  
✅ 应该是：`"冒充电商客服"`

### 错误 3：全角/半角混用
❌ 数据库：`"虚假投资理财"` （全角空格）  
✅ 应该是：`"虚假投资理财"`
