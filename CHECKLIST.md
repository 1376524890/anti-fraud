## 🐛 问题排查

### 如果某个分类显示为空：
1. [ ] 查看云函数日志，确认接收的 category 参数
2. [ ] 运行 `checkArticleCategories` 查看该分类的统计
3. [ ] 检查数据库中该分类的文章是否存在
4. [ ] 检查分类名称是否完全匹配（注意空格、标点）

### 如果所有分类都显示为空：
1. [ ] 检查云函数是否正确部署
2. [ ] 检查 `app.js` 中的云开发环境 ID
3. [ ] 检查数据库 articles 集合的权限设置
4. [ ] 查看云函数错误日志

### 如果分页加载失败：
1. [ ] 查看云函数日志，确认 page 参数
2. [ ] 检查前端 `currentPage` 是否正确递增
3. [ ] 检查 `hasMore` 状态是否正确
